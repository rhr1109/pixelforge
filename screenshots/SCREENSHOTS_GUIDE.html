1
홈 / 랜딩 페이지
핵심엔터테인먼트상업성
앱 진입 시 첫 화면. 히어로 섹션에서 플랫폼 가치를 전달하고, 트렌딩 에셋·인기 게임·주간 리더보드로 즉각적인 탐색을 유도합니다.
  • 히어로 섹션: 대장장이 마스코트(Forgy) 플로팅 애니메이션 + "상상을 픽셀로, 픽셀을 게임으로" 그라데이션 타이포
  • 실시간 통계: 에셋 1,247개 / 게임 384개 / 크리에이터 8,923명 / 다운로드 52.4K 표시
  • 트렌딩 에셋 4개: 카드 호버 시 리프트 + 그림자 효과, FREE/POPULAR/NEW/HOT 배지
  • 인기 게임 3개: 호버 시 플레이 오버레이 출현, 장르 태그, 플레이수/별점/댓글수 표시
  • 주간 리더보드: 에셋·게임 부문 2열 병렬, 금은동 배지 컬러링
🏠index.html을 브라우저에서 열면 이 화면이 표시됩니다
캡처: 브라우저 F12 → 1440x900 → 전체 페이지 스크린샷
2
에셋 브라우저
핵심상업성
게임 에셋을 카테고리별로 탐색하는 핵심 페이지. 9개 카테고리 필터와 정렬 기능으로 원하는 에셋을 빠르게 찾을 수 있습니다.
  • 9개 카테고리: 스프라이트, 타일셋, 오디오/BGM, UI 키트, 3D 모델, 게임 템플릿, 이펙트/VFX, 폰트 + 전체
  • 4종 정렬: 인기순, 최신순, 다운로드순, 좋아요순
  • 에셋 카드 12종: 카테고리별 고유 그라데이션 썸네일, 아이콘, 배지, 태그, 좋아요/다운로드 수
  • 좋아요 버튼: 하트 토글 + localStorage 영속 저장
  • 에셋 상세 모달: 카드 클릭 시 풀사이즈 미리보기 + 상세정보 + 무료 다운로드 버튼
🎨상단 네비게이션 "에셋" 클릭 후 카테고리 필터 테스트
테스트: "스프라이트" → "타일셋" 필터 전환, 카드 클릭하여 모달 확인
3
게임 쇼케이스
핵심엔터테인먼트
유저들이 자신의 게임을 올려 자랑하고, 다른 유저가 플레이·평가·댓글을 남기는 쇼케이스 공간입니다.
  • 장르 필터 7종: 전체, 플랫포머, RPG, 퍼즐, 슈터, 어드벤처, 캐주얼
  • 게임 카드: 호버 시 다크 오버레이 + 플레이 버튼이 스케일 업 애니메이션으로 등장
  • 게임 상세 모달: 대형 프리뷰 + 플레이 버튼 + 설명 + 사용 에셋 크레딧 + 댓글 시스템
  • 에셋 크레딧 연결: 어떤 Pixelforge 에셋으로 만들었는지 표시 → 에셋-게임 생태계
  • 댓글 시스템: 실시간 댓글 입력 및 기존 댓글 표시
🎮상단 "게임" 클릭 → 카드 호버 → 카드 클릭하여 모달 확인
핵심 포인트: 호버 시 ▶ 플레이 버튼 등장 애니메이션
4
업로드 (에셋 / 게임)
핵심상업성
에셋과 게임을 업로드하는 투인원 페이지. 탭 전환으로 에셋/게임 업로드 폼을 선택합니다.
  • 드래그 앤 드롭 업로드: 파일을 끌어놓으면 파일 리스트 표시 (이름, 크기, 삭제 가능)
  • 에셋 업로드 폼: 이름, 카테고리(8종), 라이선스(5종), 설명, 태그 입력
  • 게임 업로드 폼: 이름, 장르(6종), 외부 링크, 설명, 에셋 크레딧, 태그 입력
  • 즉시 반영: 업로드 시 에셋 브라우저/게임 쇼케이스에 실시간 추가
  • 게이미피케이션: 업로드 시 XP 획득 + 컨페티 축하 효과
📤"업로드" 클릭 → 에셋/게임 탭 전환 → 폼 작성 후 게시
테스트: 이름만 입력 후 "게시하기" → 에셋 브라우저에서 등록 확인
5
커뮤니티 피드
소셜엔터테인먼트
크리에이터들이 소식, 작업물, 팁을 공유하는 소셜 피드. 주간 랭킹, 업적, 공지사항이 사이드바에 배치됩니다.
  • 게시글 작성: 텍스트 입력 + 이미지/링크/태그 첨부 버튼 (UI)
  • 피드 카드: 유저 아바타, 시간, 콘텐츠, 이미지, 좋아요/댓글/공유 액션
  • 샘플 게시글 4개: 에셋 출시, 게임 출시, 타일셋 팁, VFX 작업과정 등 다양한 콘텐츠
  • 사이드바: 주간 랭킹 리더보드 (금은동 배지), 업적 그리드 (8종), 공지사항
  • 인터랙션: 좋아요 클릭 시 즉시 카운트 증가 + 토스트 알림
💬"커뮤니티" 클릭 → 게시글 작성 → 피드 확인
테스트: 게시글 작성 후 피드 최상단에 등록 확인
6
프로필 페이지
소셜상업성
유저의 작품 포트폴리오이자 개인 페이지. 탭으로 내 에셋/게임/좋아요/설정을 관리합니다.
  • 프로필 헤더: 그라데이션 아바타, 닉네임, 핸들, 소개글, 4대 통계 (에셋/게임/좋아요/팔로워)
  • 4개 탭: 내 에셋, 내 게임, 좋아요, 설정
  • 빈 상태 UI: 업로드 전 친근한 안내 메시지 + CTA 버튼
  • 설정 탭: 닉네임, 소개, 이메일, 포트폴리오 링크 편집
  • 팔로워 수: XP에 연동 (활동량 기반 시뮬레이션)
👤우상단 아바타 아이콘 클릭 → 프로필 진입
테스트: 에셋 업로드 후 "내 에셋" 탭에서 표시 확인
7
모달 (에셋 상세 / 게임 상세)
핵심엔터테인먼트
에셋·게임 카드 클릭 시 오버레이 모달로 상세 정보를 표시합니다. 다운로드, 좋아요, 공유 등 핵심 CTA가 집중됩니다.
  • 에셋 모달: 대형 썸네일, 태그, 좋아요/다운로드 수, 상세 설명, 라이선스/제작자/날짜, "무료 다운로드" CTA
  • 게임 모달: 대형 프리뷰, "플레이하기" CTA, 플레이수/별점/댓글, 설명, 사용 에셋 크레딧, 댓글 시스템
  • 배경 블러: 오버레이 backdrop-filter: blur 효과
  • 닫기: X 버튼 또는 오버레이 영역 클릭으로 닫기
  • 다운로드 시: 컨페티 파티클 + XP 획득 + 토스트 알림
🔍에셋 카드 또는 게임 카드 클릭하여 모달 확인
테스트: "무료 다운로드" 클릭 → 컨페티 + 토스트 확인

# Pixelforge

게임 에셋 오픈 플랫폼 — 에셋 공유 · 게임 쇼케이스 · 크리에이터 커뮤니티. 100% 정적, API Key 불필요, 클라이언트 사이드에서 동작합니다.

## 주요 화면

- **홈** — 트렌딩 에셋, 인기 게임, 주간 리더보드
- **에셋 브라우저** — 9개 카테고리 필터 + 정렬
- **게임 쇼케이스** — 장르별 필터 + 상세 모달
- **커뮤니티** — 피드, 랭킹, 업적
- **프로필** — 내 작품 포트폴리오
- **업로드** — 드래그 앤 드롭 게시

## 로컬 실행

```bash
# 정적 파일이라 아무 정적 서버나 OK
npx serve .
# 또는 index.html을 브라우저로 직접 열기
```

## 배포

Vercel 정적 호스팅용 `vercel.json` 포함. `vercel --prod` 로 배포.

## 스택

- 프레임워크 없는 Vanilla HTML/CSS/JS
- 이미지: 인라인 SVG + 실제 게임 스크린샷 PNG
- 반응형: 900 / 768 / 480px 브레이크포인트
